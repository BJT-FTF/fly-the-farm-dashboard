# Chemical Research Notes

## Key Principle
Drones = aircraft under APVMA. Any product label that permits "aerial application"
is legal for drone use. Labels that specify "helicopter only" are treated as
permitted-helicopter-caution (drones are aircraft but label wording may create
ambiguity). Products with NO aerial directions = not-permitted.

## Priority 1 — High-use drone chemicals (AU)

### Glyphosate 450/540 (Roundup PowerMAX, Weedmaster DST, etc.)
- Most widely used herbicide in AU
- Many formulations have aerial directions
- Aerial rate: typically 2–6 L/ha depending on target
- Water: 20–80 L/ha aerial
- Droplet: Medium–Coarse
- Non-selective — kills everything green
- Key drone uses: fallow knockdown, pasture renovation, fence lines, pre-plant
- APVMA: Hundreds of registered glyphosate products
- NOTE: Aquatic formulations (Biactive, Weedmaster Duo) have restrictions near water
- STATUS: READY TO ADD

### 2,4-D Amine 625/700
- Selective broadleaf herbicide, safe in grass
- Many labels have aerial directions
- Aerial rate: 1.5–3 L/ha
- Water: ≥50 L/ha aerial
- Droplet: Coarse–Very Coarse (drift-sensitive)
- Key drone uses: pasture broadleaf control, thistles, dock, capeweed
- Weeds: Paterson's curse, thistles, capeweed, dock, sorrel, fireweed, bindii
- CAUTION: High drift risk — must use coarse droplets, buffer zones
- STATUS: READY TO ADD

### MCPA 750
- Selective broadleaf, similar to 2,4-D but different spectrum
- Aerial directions on many labels
- Aerial rate: 1–2 L/ha
- Water: ≥50 L/ha
- Droplet: Coarse–Very Coarse
- Key uses: thistles, capeweed, dock in pasture
- STATUS: READY TO ADD

### Fluroxypyr 333 (Starane Advanced)
- Selective broadleaf, safe in grass
- Some labels have aerial directions
- Aerial rate: 0.5–1 L/ha
- Water: ≥50 L/ha
- Key uses: bindweed, thistles, woody seedlings in pasture
- STATUS: READY TO ADD

### Dicamba 700 (Kamba 750, Dicamba 700)
- Selective broadleaf
- Some labels have aerial directions
- Aerial rate: 0.3–0.7 L/ha
- Water: ≥50 L/ha
- CAUTION: Extreme drift/volatility risk — many states have restrictions
- Key uses: thistles, dock, hardheads in pasture
- STATUS: READY TO ADD (with strong drift warnings)

### Metsulfuron 600 WG (already partly in DB — expand coverage)
- More weeds to add: thistles, Paterson's curse, St John's wort, capeweed
- Already in DB for blackberry, mimosa — extend to pasture broadleaf weeds
- STATUS: EXPAND EXISTING

### Imazapyr 250 (Arsenal Xpress, OnDuty)
- Non-selective residual — total vegetation control
- Aerial directions on some labels
- Aerial rate: 2–4 L/ha
- Water: ≥50 L/ha
- Key uses: woody weed control, fence lines, rights-of-way, non-crop areas
- Weeds: rubber vine, parkinsonia, mesquite, mimosa, camphor laurel
- STATUS: READY TO ADD

### Aminopyralid 10 (Hotshot)
- Already referenced in DB as alt brand — needs own entries
- Aerial directions available
- Key uses: woody weeds, lantana, blackberry
- STATUS: READY TO ADD

### Hexazinone (Velpar L, Velmac G)
- Soil-active woody weed control
- Aerial (granular) for some formulations
- Key uses: woody weeds in pasture, regrowth control
- STATUS: RESEARCH NEEDED — check which formulations have aerial

- Aerial rate: 0.1–0.25 L/ha
- Water: ≥20 L/ha
- Key uses: GRT grass, African lovegrass, annual grasses in broadleaf crops

## Verification Checklist
For each chemical before adding to curated DB:
- [ ] Confirm APVMA registration is current (PubCRIS check)
- [ ] Read product label for aerial application directions
- [ ] Note minimum water volume for aerial
- [ ] Note droplet size requirements
- [ ] Note any buffer zone requirements
- [ ] Note withholding periods
- [ ] Note adjuvant requirements
- [ ] Confirm drone compatibility (aircraft = drone per APVMA)
- [ ] Cross-reference with state regulations (some states restrict specific chemicals)
